Mindustry auf dem Pi

Dieser Raspberry Pi hostet zwei Dinge – such dir aus, wie du spielen willst.

Pindustry – Koop im Browser Multiplayer

Ein kleines, eigens für diesen Pi geschriebenes Fabrik-Verteidigungsspiel, inspiriert von Mindustry: Bohrer aufs Erz setzen, Förderbänder zum Kern bauen, Türme gegen die Gegnerwellen. Alle im WLAN spielen zusammen auf derselben Karte – einfach öffnen, auch auf dem iPad (Touch-Steuerung eingebaut).

▶ Pindustry starten

Sofort im Browser spielen Einzelspieler

Mindustry Classic (die Original-Version von 2018) läuft direkt hier im Browser – nichts installieren, einfach loslegen. Funktioniert auch in Safari. Auf dem iPad erscheint automatisch ein virtueller Joystick zum Laufen (experimentell); Bauen und Zielen geht durch Antippen.

▶ Classic im Browser starten

Richtiges Multiplayer aktuelles Mindustry v8

Auf diesem Pi läuft außerdem ein echter Mindustry-Server für Koop/Multiplayer. Dafür brauchst du das Spiel auf deinem Gerät (Mac/PC/Android kostenlos, iPhone/iPad 1,99 €) – Browser reicht hier leider nicht, das kann Mindustry technisch nicht.
